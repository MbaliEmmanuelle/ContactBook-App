# Contact Book Application

A JavaScript contact management application built during the FNB App Academy Full Stack Development programme.

## Project Overview

This project allows users to view and manage contact information through a simple multi-page web application. It demonstrates front-end development, DOM manipulation and basic application structure.

## Technologies Used

- HTML
- CSS
- JavaScript

## Key Concepts Demonstrated

- DOM manipulation
- Event handling
- Multi-page navigation
- Form handling
- JavaScript logic
- Front-end application structure

## What I Learned

Through this project, I improved my understanding of how JavaScript can be used to make web pages interactive and how different files work together in a web application.
